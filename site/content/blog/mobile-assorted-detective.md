## Overview

**Assorted Detective** is a 2D hidden-object puzzle game that will challenge your brain. Players must find the requested items across increasingly complex and cluttered scenes.

**Platform:** iOS / Android
**Engine:** Unity
**Language:** C#
**Genre:** Puzzle / Hidden Object

## Key Features

- **Hidden object gameplay** with hundreds of items to find
- **Increasingly complex scenes** with layered difficulty
- **Timed challenges** for competitive players
- **Hint system** for when you're truly stuck
