## Overview

**Sipher Odyssey** is a multiplayer rogue-like shooter built for mobile platforms. Players team up in real-time co-op sessions, battling through procedurally generated stages filled with enemies, bosses, and randomized power-ups.

**Platform:** iOS / Android
**Engine:** Unity
**Language:** C#
**Role:** Senior Gameplay Engineer
**Genre:** Multiplayer Rogue-like Shooter

## Key Features

- **Real-time multiplayer co-op** with seamless matchmaking
- **Rogue-like progression** with randomized power-ups and stage layouts
- **Fast-paced shooter combat** optimized for mobile controls
- **Boss encounters** requiring coordinated team strategies
- **Replayable runs** with varied builds and synergies each session
