2048 Park

Overview

2048 Park is a collection of 2048 puzzle game variants, offering multiple game modes and fresh takes on the classic 2048 formula. Players slide numbered tiles to combine them and reach higher values.

Platform: iOS / Android Engine: Unity Language: C# Genre: Puzzle / Casual

View on App Store

Key Features

  • Multiple 2048 variants in a single app
  • Progression system unlocking new game modes
  • Clean UI with smooth tile animations
  • Score tracking and personal bests
