Clay Art

  • Unity
  • C#
  • Mobile
  • ASMR

Overview

Clay Art is an ASMR-style mobile game where players arrange colored parts onto a clay cube and enjoy the satisfying feeling of slicing through them. The game focuses on tactile, relaxing gameplay.

Platform: iOS / Android Engine: Unity Language: C# Genre: ASMR / Casual

Key Features

  • Satisfying ASMR cutting mechanics with haptic feedback
  • Colorful clay arrangements with creative patterns
  • Relaxing gameplay loop designed for stress relief
  • Multiple tools and cutting styles
