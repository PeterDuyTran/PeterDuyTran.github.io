## Overview

**Assassin** is a 3D puzzle game where players must eliminate enemies and save allies using limited bullets. Each level requires careful planning and strategic thinking to complete the objective.

**Platform:** iOS / Android
**Engine:** Unity
**Language:** C#
**Genre:** Puzzle / Strategy
